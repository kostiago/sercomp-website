UI e UX

UX é o responsável pelo planejamento e organização de como um produto funciona para que seja fácil e agradável para o usuário. E UI significa User Interface, o que geralmente se refere a interfaces gráficas com as quais as pessoas usuárias conseguem interagir, dessa forma, UI Designers trabalham com o aspecto visual de um produto digital, ou seja, o quão agradável é a estética da interface.

Quadro Kanban

Kanban é um sistema de controle e gestão do fluxo de produção em empresas e projetos que usa de cartões coloridos (post-its) e também recebe o nome de gestão visual, em razão do uso de cores como sinalizadores.

Backend NodeJS

Node.js é um ambiente de execução JavaScript que permite executar aplicações desenvolvidas com a linguagem de forma autônoma, sem depender de um navegador. Com ele, é possível criar praticamente qualquer tipo de aplicações web, desde servidores para sites estáticos e dinâmicos, até APIs e sistemas baseados em microserviços.

Testes de Software

O teste de software é uma maneira de avaliar a qualidade da aplicação e reduzir o risco de falha em operação. Testar não consiste apenas em executar testes (executar o software e verificar os resultados). Executar testes é apenas umas das atividades. Planejamento, análise, modelagem e implementação dos testes, relatórios de progresso, resultado e avaliação da qualidade, também são partes de um processo de testes.

Pesquisa Bibliográfica

Pesquisa bibliográfica consiste na etapa inicial de todo o trabalho científico ou acadêmico, com o objetivo de reunir as informações e dados que servirão de base para a construção da investigação proposta a partir de determinado tema.

Introdução ao Scrum

Scrum é um conjunto de boas práticas empregado no gerenciamento de projetos complexos, em que não se conhece todas as etapas ou necessidades. Ela é aplicada a partir de ciclos rápidos, chamados sprints, nos quais há um tempo determinado para que as atividades sejam concluídas, geralmente, entre duas e quatro semanas.

Desenvolvimento de Planinhas Excel

O Excel é um software que permite realizar tarefas contabilísticas e financeiras graças às suas aplicações para criar e trabalhar com folhas de cálculo.

Oficinas

Elixir

Elixir é uma linguagem de programação funcional, com o diferencial de seu código ser executado através de processos isolados. E está por trás de projetos como o WhatsApp e Discord. Além de ser uma linguagem de programação brasileira.

Artigo Científico e Projeto de Pesquisa

Projeto de pesquisa compreende toda a etapa de planejamento que se pretende seguir durante a pesquisa. Já o artigo científico é a publicação com autoria declarada, que apresenta e discute ideias, métodos, técnicas, processos e resultados nas diversas áreas do conhecimento.

VueJS

O Vue.js é um framework Javascript usado para o desenvolvimento web, usado para construir SPA (Single Page Applications) e interfaces de usuário, tornou-se uma excelente opção, também, pelo fato de ter componentes reutilizáveis e proporcionar o desenvolvimento ágil.
